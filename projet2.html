Projet Réseau pour l'entreprise GSB

Description du projet

GSB est un groupe pharmaceutique français fictif utilisé dans le cadre d’un projet pédagogique.

J’ai conçu une infrastructure réseau complète pour l’entreprise GSB, incluant VLANs, routage inter-VLAN, configuration d’équipements Cisco, plan d’adressage IP structuré et schéma de câblage.

Objectifs principaux

  • Mettre en place un réseau fiable et sécurisé
  • Segmenter les services avec des VLANs
  • Gérer l’accès invité et le Wi-Fi
  • Définir un plan d’adressage IPv4 organisé

Résultats obtenus

L’infrastructure a été modélisée sur Cisco Packet Tracer. Elle permet une gestion centralisée, une sécurité accrue grâce aux politiques VLAN et une documentation complète du déploiement.
