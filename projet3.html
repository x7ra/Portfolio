Script Batch de Gestion Réseau - Projet de Stage

Description du projet

Durant mon stage, j'ai développé un script batch complet pour automatiser la configuration des partages réseau et l'installation des imprimantes dans un environnement Windows. Ce projet répondait à un besoin concret de l'entreprise pour simplifier les déploiements et réduire les erreurs de configuration.

Objectifs principaux

  • Automatiser la configuration des partages réseau avec gestion des identifiants
  • Simplifier l'installation des imprimantes réseau avec les bons pilotes
  • Créer une interface utilisateur intuitive en ligne de commande
  • Gérer les erreurs et fournir un retour clair à l'utilisateur

Fonctionnalités clés

  • Menu interactif avec validation des saisies
  • Gestion de 30 partages réseau différents avec identifiants sécurisés
  • Installation de 24 imprimantes avec leurs pilotes spécifiques
  • Création automatique de raccourcis sur le bureau
  • Vérification de la connectivité réseau avant configuration

Technologies utilisées

Batch Script Windows CMD PowerShell Réseau TCP/IP

Résultats obtenus

Le script a permis de réduire le temps de configuration d'un poste de travail de 30 minutes à moins de 2 minutes, tout en éliminant les erreurs humaines. Il a été déployé sur l'ensemble du parc informatique et intégré dans les procédures d'onboarding.

Extraits du code
